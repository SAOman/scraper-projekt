# Scraper cytatów

Aplikacja do automatycznego zbierania cytatów ze strony quotes.toscrape.com.

## Profil danych (4 grupy)

| Grupa | Opis | Przykład |
|-------|------|---------|
| tekst | Treść cytatu | "The world as we have created it..." |
| autor | Imię i nazwisko autora | Albert Einstein |
| tagi | Lista tagów cytatu | change, thinking, world |
| link_autora | URL do profilu autora | https://quotes.toscrape.com/author/... |

## Architektura — 3 moduły (kontenery Docker)
